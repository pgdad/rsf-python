# Changelog

## 0.1.0

Initial release.

- Real-time YAML schema validation against bundled RSF JSON Schema
- Semantic validation: state references, reachability, terminal states, States.ALL ordering
- Go-to-definition for state names (Next, Default, Catch, StartAt)
- Find All References and Document Highlights
- State name autocomplete in Next/Default fields
- Quick-fix code actions for typo'd state names
- Live graph preview panel with pan, zoom, and click-to-navigate
- Status bar validation indicator
- Works without local RSF installation
