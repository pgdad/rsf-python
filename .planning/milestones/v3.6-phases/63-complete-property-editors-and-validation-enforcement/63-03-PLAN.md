  - Cause: str | None + CausePath: str | None (radio pair, debounce texts)
  - QueryLanguage: str | None (text, debounce)
  Note: Fail has NO I/O fields at all per ASL spec

ChoiceState (no _TransitionFields — uses Choices/Default instead):
  - Comment: str | None (text, debounce)
  - Choices: list[ChoiceRule] (read-only summary: "{N} rules (edit in YAML)")
  - Default: str | None (text, debounce — name of fallback state)
  - InputPath: str | None (text, debounce)
  - OutputPath: str | None (text, debounce)
  - Assign: dict | None (read-only summary)
  - Output: Any | None (read-only summary)
  - QueryLanguage: str | None (text, debounce)
  Note: No ResultPath, Parameters, ResultSelector (not from _IOFields)

ParallelState (_IOFields + _TransitionFields + _AssignOutput):
  - Comment: str | None (text, debounce)
  - Branches: list[BranchDefinition] (read-only summary: "{N} branches (edit in YAML)")
  - Retry: list[RetryPolicy] | None (read-only summary)
  - Catch: list[Catcher] | None (read-only summary)
  - End: bool toggle (from _TransitionFields, handled by transition logic but shown)
  - QueryLanguage: str | None (text, debounce)
  - I/O section: InputPath, OutputPath, ResultPath (text), Parameters, ResultSelector (read-only), Assign, Output (read-only)

MapState (_IOFields + _TransitionFields + _AssignOutput):
  - Comment: str | None (text, debounce)
  - ItemProcessor: BranchDefinition | None (read-only summary)
  - ItemsPath: str | None (text, debounce)
  - MaxConcurrency: int | None (number, immediate)
  - ItemSelector: dict | None (read-only summary)
  - Retry: list[RetryPolicy] | None (read-only summary)
  - Catch: list[Catcher] | None (read-only summary)
  - End: bool toggle
  - QueryLanguage: str | None (text, debounce)
  - I/O section: InputPath, OutputPath, ResultPath (text), Parameters, ResultSelector (read-only), Assign, Output (read-only)
