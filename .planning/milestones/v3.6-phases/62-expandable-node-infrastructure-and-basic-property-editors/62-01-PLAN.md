  <behavior>
    - toggleExpand('A') sets expandedNodeId to 'A'
    - toggleExpand('A') again sets expandedNodeId to null (toggle off)
    - toggleExpand('B') when 'A' is expanded sets expandedNodeId to 'B' (accordion)
    - updateStateProperty('A', 'Comment', 'hello') sets nodes[A].data.stateData.Comment to 'hello'
    - updateStateProperty('A', 'TimeoutSeconds', 30) sets nodes[A].data.stateData.TimeoutSeconds to 30
    - updateStateProperty('A', 'End', true) sets nodes[A].data.stateData.End to true
    - updateStateProperty on non-existent node is a no-op
    - updateStateProperty initializes stateData as {} if undefined
    - Initial expandedNodeId is null
    - removeState clears expandedNodeId if the expanded node is deleted
  </behavior>
  <action>
    Add to FlowState interface:
    - `expandedNodeId: string | null` (initial: null)
    - `toggleExpand: (nodeId: string) => void`
    - `updateStateProperty: (nodeId: string, key: string, value: unknown) => void`

    `toggleExpand` implementation:
    - If `state.expandedNodeId === nodeId`, set to null (collapse)
    - Otherwise set to nodeId (expand, auto-collapsing previous — accordion)

    - If not found, return (no-op)
    - If `node.data.stateData` is undefined, initialize as `{}`
    - Set `node.data.stateData[key] = value`
    - If value is `undefined` or `null`, use `delete node.data.stateData[key]` instead (for clearing optional fields)
